# 🛰 ROSMASTER — Yahboom ROSMASTER X3 (ROS 2)

Documentation, specifications, and demonstration media for the **Yahboom
ROSMASTER X3** — a **ROS 2** STEM robot platform used for autonomous navigation
and perception work.

> Category: **Autonomous Mobile Robotics** · Platform: **ROS 2 + depth camera + LiDAR**

---

## 🎯 Overview

The ROSMASTER X3 is a commercial ROS 2 robot platform. This folder collects its
specification, the perception hardware it is configured with, and photo/video
documentation of the physical robot.

## 🧰 Platform Specs

| Item | Detail |
|------|--------|
| **Base** | Yahboom ROSMASTER X3 (ROS 2 STEM robot) |
| **Controller** | ROS Expansion Board |
| **Depth camera** | Intel RealSense **D435** |
| **LiDAR** | Livox **Mid-360** |
| **Software** | ROS 2 |

**Reference links**
- Product page: <https://category.yahboom.net/products/rosmaster-x3>
- Demo video: <https://youtu.be/KT_r-rOF3yw>
- RealSense D435: <https://robu.in/product/intel-realsense-depth-camera-d435f/>
